About Us

House Slytherin is proud to introduce Bob's Computer Repair Shop. This website was built using the latest technology in web development both the wizarding and muggle community. This website allows users to select service they want, print an invoice, update user information, update security questions, update services, and also delete any unwanted information. This website is secured from any unwanted guests.

Owners
